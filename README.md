# 🚀 Agara NFT Marketplace

Modern NFT Marketplace built with:

- React + Vite
- thirdweb SDK
- IPFS Storage
- ERC721 NFT
- OpenSea Compatible Metadata
- Wallet Connect
- NFT Gallery UI

---

# ✨ Features

✅ Wallet Connect (MetaMask / WalletConnect)  
✅ NFT Mint Page  
✅ Automatic IPFS Upload  
✅ OpenSea Compatible Metadata  
✅ NFT Gallery Grid  
✅ Responsive UI  
✅ ERC721 Compatible  
✅ thirdweb Integration
